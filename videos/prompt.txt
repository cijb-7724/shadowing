**【厳守事項：タイムスタンプと文字起こしの処理】**
1.  **レコード単位の原則**:
    * 出力データは、動画内の**英文を正確に一文ずつ区切り**、その**一文ごとに一つのレコード（行）**を作成してください。
2.  **タイムスタンプ処理（シャドーイング最適化）**:
    * タイムスタンプは、**実際に音声を聞いた上で**、文頭からしっかり聞き取れるよう、音声の**話し出しの開始時刻**を特定してください。
    * 特定した音声開始時刻のミリ秒単位を**必ず切り捨て**て整数秒とします。
    * その整数秒から、シャドーイングのための**1秒のバッファ（遡り）を適用**し、最終的な再生開始秒として`[MM:SS]`形式で設定します。
3.  **語彙リスト**:
    * 想定学習者は**日本の中学3年生**とし、そのレベルで**初めて登場すると思われる英単語・熟語のみ**を抽出・選定し、語彙リストに含めます。

        // タイムスタンプの秒数に変換 (HH:MM:SS形式にも対応)
        function getSeconds(timeStr) {
            const parts = timeStr.replace(/[\[\]]/g, '').split(':').map(Number);
            let seconds = 0;
            if (parts.length === 3) { // HH:MM:SS
                seconds = parts[0] * 3600 + parts[1] * 60 + parts[2];
            } else if (parts.length === 2) { // MM:SS
                seconds = parts[0] * 60 + parts[1];
            }
            return seconds;
        }
